These 8 files are the 8 batch scripts and are the steps of the workflow which automatically runs tutorial 1 in full, where each job submits the next, in order.

To launch this workflow, 

o change the username and project code to your username and project code, 
o copy all 8 files into the parent (Ketton) directory above
o change directory to the parent direcorty: cd ..
o launch the workflow but submitting the first batch script: sbatch workflow_step_1.bat
